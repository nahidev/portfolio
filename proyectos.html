Bookshelf

Es una estantería virtual desarrollada con Java, HTML, JavaScript y CSS, utilizando los frameworks Vue y Spring Boot.

La aplicación permite llevar un registro de todos los libros que lees a lo largo del año, seguir tu progreso de lectura, calificar cada libro y descargar tu historial completo de lecturas.

Memory Game

Este proyecto es un juego clásico de memoria en los lenguajes HTML, CSS y JavaScript.

Las cartas se giran al hacer clic y permanecen bloqueadas si coinciden. Se completa el juego al encontrar todos los pares de cartas.

Snake Game

Este proyecto es un juego clásico de Snake en el lenguaje Java.

Incrementa la velocidad a medida que aumentan las manzanas atrapadas, con manzanas doradas ocasionales y niveles de dificultad.

Science Quiz

Este proyecto es un cuestionario desarrollado utilizando los lenguajes HTML, CSS y JavaScript.

Ofrece la posibilidad de elegir preguntas de geografía o ciencia, con cuatro respuestas posibles. Al seleccionar una respuesta, el sistema indica si la respuesta es correcta o incorrecta. Al final, muestra la puntuación. Permite reiniciar el cuestionario randomizando las preguntas.
